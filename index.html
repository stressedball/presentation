Un site pour un commerce

Description

Projet réalisé avec la librairie React. L'idée étant de créer des composants qui seront utilisés pour générer les éléments s'affichant sur la page, ainsi que la gestion des données et navigation côté client. L'idée est de simuler un site marchand où le visiteur peut ajouter des articles à son panier.

Un jeu de mémoire

Description

Un jeu de "carte mémoire". Le but est simple. Vous avez 5 niveaux pour terminer le jeu. A chaque niveau, vous devrez cliquer sur toutes les cartes une fois et seulement une fois. Cela parait simple mais méfiez-vous l'exercice est plus compliquée qu'il n'y parait. Réalisé avec React également.
